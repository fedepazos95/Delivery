# Delivery App

Delivery app with functions to add, edit and delete deliveries.

#### Getting Started
```
> git clone https://github.com/fedepazos95/Delivery.git
> cd Delivery
> yarn install
> yarn start
```
## Built With

* [React](https://reactjs.org/)
* [react-bootstrap](https://react-bootstrap.github.io/)
